# 🎭 RoleChat-AI

AI-powered role-based chatbot built with Streamlit, Groq API, and Llama 3 models.

🔗 [Live Demo](https://rolechat-ai-nc8ummnluwpmhf9nhdvzd2.streamlit.app/)

✨ Features

- AI-powered conversational chatbot
- Role-based AI interaction
- Ultra-fast responses using Groq inference
- Llama 3 model integration
- Clean and modern Streamlit UI
- Secure API key handling
- Cloud deployment support

🛠️ Used Technologies

- Python
- Streamlit
- Groq API
- LangChain
- Llama 3

📄 Example Interaction

```text
User: Act like a fitness coach

AI: Great! Let's start your fitness journey 💪
What's your current goal:
- Weight loss
- Muscle gain
- Better stamina
```
